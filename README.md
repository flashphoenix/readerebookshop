# Readers Bookshop E-commerce Website

This is a professional e-commerce website for Readers Bookshop, a Kenyan business specializing in school textbooks and educational supplies. The website is designed to be visually appealing, user-friendly, and optimized for parents, students, and educators in Kenya.

## Features

- Responsive design that works on all devices (mobile, tablet, desktop)
- Professional and engaging visual design with brand-consistent colors
- Parallax scrolling effects and smooth animations
- Interactive product categories and filtering
- Special offers sections with auto-scrolling gallery
- Detailed product pages with image gallery
- Contact form with validation
- About Us page with engaging "opening book" animation

## Pages

1. **Home** - Features a hero section, company overview, special offers, product categories, and delivery service promotion
2. **About Us** - Detailed information about Readers Bookshop with an engaging animation
3. **Store** - Product listings with filtering by category, special offers, and new products
4. **Product Detail** - Comprehensive product information with images, specifications, and related products
5. **Contact** - Contact form and information for reaching out to the bookshop

## Technologies Used

- HTML5
- CSS3 (with modern features like flexbox, grid, and animations)
- JavaScript (vanilla, no frameworks)
- Font Awesome icons
- Google Fonts (Poppins and Montserrat)

## Installation and Usage

1. Download the `readers_bookshop_website.zip` file
2. Extract all files to a directory on your computer
3. Open the `index.html` file in a web browser to view the website
4. To deploy to a live server, upload all files to your web hosting service

## File Structure

- `index.html` - Home page
- `about.html` - About Us page
- `store.html` - Store/Products page
- `contact.html` - Contact page
- `product.html` - Product detail page template
- `styles.css` - All styles for the website
- `script.js` - JavaScript functionality
- Image files:
  - `logo.png` - Readers Bookshop logo
  - `hero_image.png` - Hero section background
  - `special_offers.png` - Image for special offers section
  - `about_us.png` - Image for the About Us page
  - `categories.png` - Image for product categories
  - `delivery_service.png` - Image for delivery service section

## Customization

- Colors can be adjusted in the `:root` section of the CSS file
- Images can be replaced with actual photos of the bookshop and products
- Product information can be updated with real inventory data
- Contact information should be updated with the correct details for Readers Bookshop

## Browser Compatibility

The website is compatible with modern browsers including:

- Firefox (latest)
